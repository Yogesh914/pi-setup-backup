# pi-setup-backup

Backup package for my Pi extensions, skills, prompts, and themes.

## Install

```bash
pi install git:github.com/Yogesh914/pi-setup-backup
```

Or from a local clone:

```bash
pi install /path/to/pi-setup-backup
```

## Contents

- `extensions/` - Pi extensions
- `skills/` - Pi skills
- `prompts/` - Pi prompt templates
- `themes/` - Pi themes
